# pokedex-app
Pokedex project pulling and working with api. 
Pokédex API App

A responsive web application that allows users to search and explore Pokémon using real-time data from the PokéAPI. This project demonstrates strong fundamentals in API integration, asynchronous JavaScript, and dynamic DOM manipulation.

Features
🔍 Search for Pokémon by name
📊 Display key data (image, stats, type, abilities)
⚡ Real-time data fetching from an external API
📱 Responsive design using Bootstrap
❌ Error handling for invalid searches
Tech Stack
HTML5
CSS3 / Bootstrap
JavaScript (ES6)
REST API (PokéAPI)
How to Use
Enter a Pokémon name in the search bar
Click search or press enter
View Pokémon details instantly
What This Project Demonstrates
Working with APIs (fetch, async/await)
Handling and displaying JSON data
DOM manipulation and event handling
Building responsive UI layouts
